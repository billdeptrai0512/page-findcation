# Findcation Design System - Component Guidelines

## Buttons
- **Shape**: Pill-shaped (`border-radius: 9999px`)
- **Primary**: Uses the primary gradient (`#E31C5F` dominant) with white text. Soft drop shadow.
- **Secondary**: Uses the secondary gradient (`#588AC8` dominant) with white text.

## Cards
- **Background**: White `#ffffff`
- **Border Radius**: `16px`
- **Shadow**: Soft drop shadow `0 4px 12px rgba(0, 0, 0, 0.05)`
- **Content**: Clean padding (typically `16px` or `24px`) with `h3` or `body` typography.

## Map Preview Container
- **Background**: Soft blue `linear-gradient(to bottom, #ffffff, #fafbfc)` or `#f0f7ff`.
- **Purpose**: Acts as a vibrant, interactive-feeling backdrop for map markers and location data.

## Trust Score Badges
- **Shape**: Fully rounded (`border-radius: 9999px`)
- **Colors**: Soft green background (`#e8f5e9`) with dark green text (`#2e7d32`).
- **Typography**: `<small>` (14px) medium or bold.

## Icon Badges
- Used inside cards or as USP highlights.
- Keep them visually lightweight, using neutral gray (`#6b7280`) or primary gradient for emphasis.
